# תוכנית שנתית תשפ"ז — ישיבת אלוני הבשן

לוח שנה אינטראקטיבי לרכז החברתי.

## הרצה מקומית
```bash
npm install
npm run dev
```

## בנייה לפרסום
```bash
npm run build
```
התיקייה `dist/` מכילה את הקבצים להעלאה לגיטהאב.

## העלאה ל-GitHub Pages
1. העלה את **תוכן תיקיית `dist/`** לגיטהאב (לא את כל הפרויקט)
2. הפעל GitHub Pages על ה-branch הרלוונטי
